FuturSample est un logiciel MacOS splitteur de stems destinés aux beatmakers voulant resampler leur prods.

Il suffit de sélectionner le dossier contenant toutes les prods au format audio (formats supportés: .mp3, .wav, .flac, .ogg) puis de lancer le traitement.

Projet testé sur MacBook Air M3, 16 Go de RAM. (Demucs est gourmand)
